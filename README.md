# TeraDpsMeterData
Data for Tera DPS meter

Skills format:
--------------------
Skill id [tab] Global Skill name [tab] Specific Skill Name [tab] Chained/Unchained

Monsters format:
-------------------
Monster Zone ; Monster id ; Monster name

Zone format:
------------------
Monster Zone ; Zone name
